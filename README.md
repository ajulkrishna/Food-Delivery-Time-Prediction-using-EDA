# Food Delivery Time Prediction using Exploratory Data Analysis (EDA)

## 📌 Overview
This project focuses on analyzing food delivery data to understand the factors affecting delivery time. Using Exploratory Data Analysis (EDA), the project identifies key patterns, trends, and relationships that influence delivery efficiency.

## 🎯 Objectives
- Analyze delivery time patterns across different conditions
- Identify key factors affecting delivery duration
- Perform data cleaning and preprocessing
- Generate insights using visualizations

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Key Analysis Performed
- Handled missing values and data inconsistencies
- Explored relationships between delivery time and factors like:
  - Distance
  - Weather conditions
  - Traffic density
  - Order type
- Visualized trends using graphs and charts
- Performed feature-level analysis to identify major influencing factors

## 📈 Key Insights
- Delivery time increases significantly with higher traffic conditions
- Weather conditions have a noticeable impact on delivery delays
- Distance is a primary factor influencing delivery duration
- Certain order types require longer preparation and delivery time

## 📁 Project Structure
- `Project.ipynb` → Contains complete EDA and analysis

## 🚀 How to Run
1. Clone or download this repository
2. Open `Project.ipynb` using Jupyter Notebook
3. Run all cells to view analysis and results
